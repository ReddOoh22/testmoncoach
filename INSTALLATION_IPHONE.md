# Installation iPhone

1. Publier le dossier avec GitHub Pages.
2. Ouvrir l'URL dans Safari sur iPhone.
3. Toucher le bouton Partager.
4. Choisir Sur l'ecran d'accueil.
5. Ajouter l'app.

Si l'ancienne version reste en cache, supprimer l'icone de l'ecran d'accueil, recharger l'URL dans Safari, puis ajouter a nouveau l'app.
