# Mon Coach - version 1.4

Application PWA personnelle pour suivre des seances sportives sur iPhone.

## Nouveautes v1.1

- Ecran d'accueil de premiere utilisation.
- Gestion de plusieurs profils locaux.
- Questionnaire complet de creation de programme.
- Generation locale de programme sans IA et sans connexion.
- Historique separe par profil.
- Bibliotheque d'exercices.
- Ajout d'exercices personnalises.
- Creation de seances personnalisees depuis la bibliotheque.
- Export et import complet des donnees.

## Installation GitHub Pages

Envoyer tout le contenu du dossier a la racine du repository GitHub Pages.

Fichiers importants a la racine :

- index.html
- style.css
- app.js
- data.js
- manifest.webmanifest
- service-worker.js
- assets/
- icons/
- .nojekyll

## Donnees

Les donnees restent dans le stockage local du navigateur de l'appareil.
Utiliser Reglages > Exporter tous les profils pour sauvegarder.


## Version 1.2

- Retour automatique a la selection de profil au lancement/reprise de l'app.
- Suppression d'un profil individuel sans effacer les autres profils.
- Conservation des donnees locales existantes de la V1.1.


## Version 1.3

- Ecran de selection des profils enrichi.
- Resume par profil : prochaine seance, historique et progression hebdomadaire.
- Accueil du profil plus lisible avec statistiques rapides.

## Version 1.4

- Ecran de preparation avant chaque seance.
- Mode Forme normale : seance complete.
- Mode Fatigue moyenne : maximum 2 series par exercice et repos plafonnes.
- Mode Journee chargee : bascule vers la version urgence si disponible, sinon volume minimal.
- L'etat choisi est enregistre dans l'historique de la seance.
