# Mon Coach - version 1.1

Application PWA personnelle pour suivre des seances sportives sur iPhone.

## Nouveautes v1.1

- Ecran d'accueil de premiere utilisation.
- Gestion de plusieurs profils locaux.
- Questionnaire complet de creation de programme.
- Generation locale de programme sans IA et sans connexion.
- Historique separe par profil.
- Bibliotheque d'exercices.
- Ajout d'exercices personnalises.
- Creation de seances personnalisees depuis la bibliotheque.
- Export et import complet des donnees.

## Installation GitHub Pages

Envoyer tout le contenu du dossier a la racine du repository GitHub Pages.

Fichiers importants a la racine :

- index.html
- style.css
- app.js
- data.js
- manifest.webmanifest
- service-worker.js
- assets/
- icons/
- .nojekyll

## Donnees

Les donnees restent dans le stockage local du navigateur de l'appareil.
Utiliser Reglages > Exporter tous les profils pour sauvegarder.
